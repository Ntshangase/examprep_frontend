#officail documentation for exam system application.

## Naming Convesion
- All component names should follow `PascalCase`.
- All component names are intended to be descriptive, to give meaning and convey purpose of the component.
- AllFolders follow PascalCase naming convention, to tell JSX that these are custom components, since JSX treats folders names starting with small letter as native html.

## Components
- This project uses `functional components`. they are recommended components in new react versions, and are easier to read, and write, as there is less boilerplate code and allow for seamless state handling through useEffect and useState Hooks.
